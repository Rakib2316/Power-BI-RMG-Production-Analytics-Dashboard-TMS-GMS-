# 🏭 RMG Production Analytics Dashboard (TMS + GMS)

## 📌 Project Overview

Developed an interactive **Power BI dashboard** to analyze **Textile Manufacturing System (TMS)** and **Garments Manufacturing System (GMS)** production data. The dashboard helps monitor production efficiency, quality performance, factory productivity, finishing operations, and shipment performance through dynamic KPIs and visual analytics.

---

## 🎯 Project Objectives

- Analyze textile and garments production performance.
- Build an optimized star schema data model.
- Track manufacturing KPIs across multiple production stages.
- Compare factory performance using interactive filters.
- Identify production bottlenecks and quality issues.
- Generate actionable business insights for decision-making.

# 📊 Dashboard 1 — Textile Manufacturing System (TMS)

### KPIs

- Knitting Output (KG)
- Average Knitting Efficiency (%)

---

## 💡 Key Business Insights

- Knitting efficiency showed continuous improvement over the analyzed years.
- Dyeing reprocessing rate differs significantly among fabric types.
- Fabric inspection results highlight major quality defect categories.
- GSM variation before and after finishing remains relatively stable.
- Cutting achievement is consistently close to production targets.
- Sewing efficiency varies across factories.
- Production quantity differs by factory capacity.
- Finishing output does not always reach target levels.
- Shipment performance indicates opportunities to improve on-time delivery.
- Interactive slicers enable flexible performance analysis across factories, months, and years.
